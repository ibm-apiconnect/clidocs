# apic consumer-org-settings:update

Update the Consumer Organization Setting object

## Synopsis

Update the Consumer Organization Setting object
          
          Fields allowed but ignored:
          - name
          - type
          - api_version
          - scope
          - created_at
          - updated_at
          - url

```
apic consumer-org-settings:update [flags]
```
