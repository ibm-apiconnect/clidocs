# apic monitor

Monitor your APIs

## Synopsis

Run your API Tests repeatedly to identify issues.

```
apic monitor --mode consumer [flags]
```

## Options

```
  -f, --frequency int   Frequency to run tests (seconds) (default 10)
  -h, --help            Help for monitor
```
